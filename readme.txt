Seriality
Serial Connectivity Plug-in

To Compile (Requires GCC4+)
% cd seriality
% scons

To Clean
% cd seriality
% scons -c

To Package for OSX:
¥ Rename 'libseriality.dylib' to 'Seriality'
¥ Show contents of the .plugin bundle from the .dmg distribution
¥ Browse to the MacOS folder and replace 'Seriality' with new version
